## Getting Started
To get a local copy of the repository please run the following commands on your terminal:
- ```$ cd <folder>```
- ```$ git clone https://github.com/medobarakat/movie-search.git ```
- ```$ cd movie-search ```
- ```npm install```
- ```npm start```
